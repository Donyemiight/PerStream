How it works

01

Creator uploads

Drop an MP3, set a price per second. Default 0.0003 USDC ≈ $1.08/hr.

02

Listener signs in

One-click email login. No MetaMask, no seed phrase. Embedded wallet via Circle Agent Stack.

03

Press play

Every second settles a gasless USDC micro-payment on Arc via x402 + Nanopayments.

04

Creator earns

Live earnings in the dashboard. From listener #1, no audience minimum.
